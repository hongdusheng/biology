# 🧬 智能生物知识网络思维导图生成器（AI驱动版）

一个具备AI推理能力的专业思维导图生成工具，能够将生物知识文本转化为智能关联、动态适配的交互式思维导图，让思维导图成为"能思考的知识导航图"。

## 🌟 AI智能特性（新增）

### 1. 隐性关联挖掘
- **自动发现深层逻辑关系**：不仅展示显性的父子层级，还能挖掘隐藏的逻辑关联
  - 示例：线粒体 → 主动运输（ATP供能关系）
  - 示例：线粒体 → 细胞凋亡（细胞色素c信号）
- **关联依据标注**：每条隐性关联都附带推理依据
- **优先级分级**：high/medium/low三级优先级，突出重要关联

### 2. 动态层级适配
- **学段智能识别**：
  - 初中：2层深度，基础概念（如"光合作用场所、原料、产物"）
  - 高中：4层深度，详细过程（如"光反应→类囊体膜→ATP生成"）
  - 大学：6层深度，分子机制（如"光系统II→P680→电子传递链"）
- **自动过滤不适内容**：根据学段排除超纲知识点

### 3. 易错点智能识别
- **常见错误标注**：自动识别并高亮易错概念
- **错误对比展示**：
  - ❌ 错误：食物链可以从消费者开始
  - ✅ 正确：食物链起点必须是生产者
  - 💡 示例：草→兔→狐（正确）；兔→狐（错误）
- **脉冲动画提示**：易错节点带有视觉提醒

### 4. 智能推荐系统
- **点击即推荐**：选中任意节点，自动推荐5个高价值相关节点
- **推荐类型**：
  - 直接关联：父子、兄弟节点
  - 隐性关联：深层逻辑关系
  - 同层概念：横向对比学习
- **学习优先级**：每个推荐带有优先级标签（基础级/重点级/拓展级）

### 5. 易混淆概念对比
- **自动生成对比链接**：如"极体 vs 极核"
- **差异表格展示**：
  - 来源：极体来自卵原细胞减数分裂 vs 极核来自胚囊细胞有丝分裂
  - 染色体数：n vs 2n
  - 功能：退化消失 vs 参与受精形成胚乳
- **常见误区提示**：标注学生易犯的混淆错误

## ✨ 核心功能

### 1. 概念提取精度
- 从生物教材章节、课程大纲或知识点文本中精准提取核心概念
- 自动识别细胞器、生理过程、遗传规律等生物学术语
- 每个概念包含：唯一ID、名称、简明定义、所属知识模块

### 2. 层级结构逻辑
- 严格遵循生物知识的逻辑层级
- 结构：学科总纲 → 模块 → 单元 → 核心概念 → 子概念 → 关键特征
- 示例：生物学 → 分子与细胞 → 细胞的基本结构 → 细胞器 → 线粒体 → 功能：有氧呼吸主要场所

### 3. 关联关系标注
识别并标注概念间的横向关联：
- **因果关系**：如"基因突变 → 生物进化"（标注"根本原因"）
- **对比关系**：如"有丝分裂 vs 减数分裂"（标注"异同点"）
- **协同关系**：如"光合作用 ⇄ 呼吸作用"（标注"能量转化互补"）

### 4. 交互式可视化
- 节点可点击查看详细信息
- 支持展开/折叠层级结构
- 缩放、平移操作
- 关联线动态显示

## 📖 使用指南

### 1. 配置学习参数（新增）
- **选择学段水平**：初中/高中/大学（影响知识深度）
- **输入薄弱知识点**（可选）：如"有丝分裂后期、光合作用暗反应"
- **启用隐性关联挖掘**：勾选后自动发现深层逻辑关系

### 2. 输入知识文本
在输入框中输入生物知识文本，例如：
```
细胞是生物体的基本结构和功能单位。细胞由细胞膜、细胞质和细胞核组成。
细胞质中含有多种细胞器，如线粒体、叶绿体、核糖体等。
线粒体是有氧呼吸的主要场所，叶绿体是光合作用的场所。
```

### 3. 选择知识模块
从下拉菜单中选择对应的生物学模块：
- 分子与细胞
- 遗传与进化
- 稳态与调节
- 生物与环境

### 3. 使用示例模板
点击示例按钮快速加载预设的知识文本：
- 细胞结构
- 细胞分裂
- 遗传基础
- 光合作用

### 4. 生成智能思维导图
点击"生成思维导图"按钮，AI系统将自动：
- 提取核心概念并识别易错点 ⚠️
- 构建层级结构并适配学段深度 🎓
- 识别显性和隐性关联关系 🔗
- 计算学习优先级（基础级/重点级/拓展级）📊
- 生成易混淆概念对比 🔄
- 渲染可视化图谱 🎨

### 5. 智能交互操作（AI增强）
- **点击节点**：查看详细信息 + 获取智能推荐
  - 易错点提示：显示常见错误和正确答案
  - 智能推荐：推荐5个高价值相关节点
  - 优先级标记：节点左上角显示"基/重/拓"
- **推荐跳转**：点击推荐节点名称快速跳转并聚焦
- **关联可视化**：
  - 实线：显性关联（父子、兄弟）
  - 虚线：隐性关联（深层逻辑）
  - 对比线：易混淆概念对比
- **点击 +/- 按钮**：展开/折叠子节点
- **拖拽画布**：平移视图
- **工具栏按钮**：放大、缩小、重置视图、全部展开/折叠
